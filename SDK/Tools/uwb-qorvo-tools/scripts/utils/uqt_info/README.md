# uqt_info

Print environment information

## Example

```
uqt_info

# Python3 version:
    ...
# Customization (UQT_CUSTOM):
    ....
# Wanted Extensions (UQT_ADDINS):
    addin_name_1
    addin_name_2
    addin_name_3
    ...
# Loaded Extensions:
    ...
# DUT 'Default':
    unknown port ('UQT_PORT' not defined.)
```
